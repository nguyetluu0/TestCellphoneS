# Giới thiệu

Dự án thực hiện kiểm thử tự động cho website CellphoneS sử dụng Selenium WebDriver

# Cách chạy project

1. Clone project
2. Mở bằng IntelliJ IDEA
3. Cài đặt package cần thiết
4. Run test
